# Flight-Booking-Management-System
Designed and implemented a MySQL-based flight booking database to manage flights, passengers, bookings, and ticket information using relational database concepts and SQL queries.

## Overview
A MySQL database project designed to manage airlines, flights, passengers, bookings, and ticket information.

## Features
- Flight Scheduling
- Passenger Management
- Booking System
- Revenue Analysis
- SQL Queries and Joins

## Technologies Used
- MySQL
- SQL
